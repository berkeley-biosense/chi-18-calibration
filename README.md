# usenix-17-stimuli

PsychoPy stimuli for our USENIX '17 submission.

designed by @[mtcurran](https://github.com/mtcurran).

## installation

1. [Download PsychoPy](http://www.psychopy.org/) if you don't already have it.
2. Download and run our [OpenBCI collector](https://github.com/berkeley-biosense/openbci-collector), and run it on port 3000.
3. In PsychoPy, open up the "experiment.psyexp" file and hit the green "Run" button to start it.
